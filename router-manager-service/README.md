## Router Manager Service

Сервис управления командами для роутеров. Предоставляет gRPC API для отправки команд роутерам, получения команд и подтверждения их выполнения.

## Обзор

Router Manager Service — это приложение, которое:

- Отправка команд одному или всем роутерам
- Хранение статусов выполнения команд в PostgreSQL
- Получение команд роутерами по запросу
- Подтверждение выполнения команд роутерами

## Технологии

- **Go 1.20+** - Язык программирования
- **Postgres** - База данных
- **Testcontainers** - Фреймворк для интеграционного тестирования

# Инструкция по локальному запуску

```bash
make help
```

##  Запуск необходимой инфраструктуры
Откройте .env.example в редакторе, измените пароли/логины по желанию

```bash
make router-manager-service-infra-up
```

#  Проверка работы
```bash
make router-manager-service-infra-check
```

# Запуск приложения

Для запуска сервиса при необходимости нужные установите переменные окружения или используйте дефолтные

Запустите unit и интеграционные тесты
```bash
make router-manager-service-app-test
```

Запустите приложение
```bash
make router-manager-service-app-run
```
